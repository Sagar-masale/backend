Backend Overview

This is the backend service for PBS Jewellers built using Node.js, Express.js, and MongoDB. It handles authentication, product management, order processing, admin operations, and REST APIs for the frontend application.

Backend Structure:
backend/
│
├── src/
│   ├── controllers/
│   ├── models/
│   ├── routes/
│   ├── middleware/
│   ├── database/
│   └── utils/
│
├── server.js
└── package.json
